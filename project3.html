CONCEPT

Facebook is a comment social medial, people use it everyday and like facebook page. When those page are getting more and more, it's difficult to check every pages you like.

We separate it by each category, user can easily check all the pages that user liked. Click Timeline button, user can see a chart that is user liked those pages from the first page until now. Click Friends button, user can check how many pages that you liked with your friends if user's friends also use this website.
